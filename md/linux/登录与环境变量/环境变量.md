## 系统环境变量
* 名称
    * PATH
        * 含义:命令文件夹所在的路径
        * 书写方式:text。每个文件夹使用`:`分隔 
        * 读写:读写
    * HOME
        * 含义:用户家目录
        * 书写方式:text。文件夹路径
        * 读写:读写
    * SHELL
        * 含义:当前用户的shell解释器
        * 书写方式:text。解释器路径
        * 读写:读写
    * HISTSIZE
        * 含义:`~/.bash_history`能保存的历史命令的条数 
        * 书写方式:int
        * 读写:读写
    * LANG
        * 含义:当前用户的语言环境
        * 书写方式:text。例如:LANG=en_US.UTF-8
        * 读写:读写
    * USER
        * 含义:当前用户的用户名
        * 读写:只读 
    * PWD
        * 含义:当前的工作目录
        * 读写:只读 
    * HOSTNAME
        * 含义:主机名
        * 读写:只读
* 配置方式:在配置文件中写入赋值文本,一行一条。示例如下
    ```shell
    SHELL=/bin/bash
    ```

## 配置文件 
* 全局配置文件
    * `/etc/environment`
        * 格式:纯文本,一行一句文本,且只能写为`KEY_NAME="VALUE"`形式,如`PATH="/usr/bin"`。可用`#`注释
        * 用途:设置全局环境变量
    * `/etc/profile`
        * 格式:shell脚本
        * 调用的脚本
            * `/etc/bash.bashrc`或`/etc/bashrc`
            * `/etc/profile.d/*.sh`,即`/etc/profile.d/`文件夹下所有的后缀为`.sh`的文件
        * 用途:详见`登录`
* 用户配置文件 
    * BASH_PROFILE
        * 格式:shell脚本,它表示以下三个文件中按顺序第一个存在的文件
            * `~/.bash_profile`
            * `~/.bash_login`
            * `~/.profile` 
        * 调用的脚本
            * `~/.bashrc`
        * 用途:详见`登录`
    * `~/.bashrc`:shell脚本 
        * 格式:shell脚本
        * 用途:详见`登录`
    * `~/.bash_logout`
        * 格式:纯文本,一行一句文本。每一行记录一条命令
        * 用途:详见`登录`
    * `~/.bash_history`
        * 格式:shell脚本
        * 用途:当前用户历史输入过的命令
