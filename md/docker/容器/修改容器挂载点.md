## 容器新增/修改挂载点
* 相关文件及对应项的值的格式
    * 文件路径:`/var/lib/docker/containers/CONTAINER_ID/hostconfig.json`,`CONTAINER_ID`是容器的完整ID 
        * 键:`Binds`
        * 值格式
            ```json
            [
                "HOST_DIR:CONTAINER_DIR"
            ],
            ```
        * 值含义
            * `HOST_DIR`,主机的被挂载到容器的目录
            * `CONTAINER_DIR`,容器挂载主机目录的目录
    * 文件路径:`/var/lib/docker/containers/CONTAINER_ID/ config.v2.json`,`CONTAINER_ID`是容器的完整ID 
        * 键:`MountPoints`
        * 值格式
            ```json
            {
                "CONTAINER_DIR": {
                    "Source": "HOST_DIR",
                    "Destination": "CONTAINER_DIR",
                    "RW": true,
                    "Name": "",
                    "Driver": "",
                    "Type": "bind",
                    "Propagation": "rprivate",
                    "Spec": {
                        "Type": "bind",
                        "Source": "HOST_DIR",
                        "Target": "CONTAINER_DIR"
                    },
                    "SkipMountpointCreation": false
                }
            }
            ```
        * 值含义
            * `HOST_DIR`,主机的被挂载到容器的目录
            * `CONTAINER_DIR`,容器挂载主机目录的目录
* 新增/修改挂载点的步骤
    1. 停止所有容器 
    1. 停止docker服务
    1. 修改`hostconfig.json`文件 
    1. 修改`config.v2.json`文件 
    1. 启动docker服务 
    1. 启动容器
