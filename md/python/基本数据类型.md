# 基本数据类型
* 数字
    * 注意
        * python3.6之后的版本的数字字面值除`0b`,`0o`,`0x`之前和它们之间、最后一个数字之后、小数点前后，可在任意位置间加`_`,不影响数字的值大小 
    * 类型
        * int 
            * 含义:整数
            * 取值范围：无限
            * 正负:`+-`
            * 字面值
                * 十进制:9,999_999
                * 二进制:0b1001,0B1001,0B1_001
                * 八进制:0o11,0O11
                * 十六进制:0x9,0X9 
            * 构造整数的方法
                * 使用字面值赋值。如执行`a=99_09`后`a`的值为`9909`
                * `int([x=0])`
                    * 含义:构造整数。若传入参数为小数,则只保留整数部分
                    * x:十进制数字或十进制数字的字符串
                    * 返回:整数
                * `int(x,base=10)`
                    * 含义:构造整数
                    * x:表示整数数字的字符串 
                    * base:x的进制。默认为10 
                    * 返回:整数
        * float
            * 含义:浮点数
            * 取值范围：可在`sys.float_info`变量中查看
                * `windows10 64bit`：[2.2250738585072014e-308,1.7976931348623157e+308] 
            * 正负:`+-`
            * 有效数字:16位
            * 字面值
                * 整数的字面值 
                * .001 
                * .0_123_4449_0
                * 3.14e-10,3.14E-10,1e10
                    * 注:总是以10进制看待底数与指数 
            * 构造浮点数的方法
                * 使用字面值赋值。如执行`a=99.09`后`a`的值为`99.09`
                * `float([x=0.0])`
                    * 含义:构造浮点数
                    * x:字面值或十进制数字的字符串或特殊含义的字符串
                        * 无穷大:`float("Infinity")`,`float("inf")`。`Infinity`和`inf`不区分大小写 
                        * 非数:`float("nan")`。`nan`不区分大小写 
                    * 返回:浮点数
        * complex
            * 含义:浮点数
            * 取值范围
                * 实部:同float
                * 虚部:同float
            * 符号
                * 实部:`+-`
                * 虚部:`+-`
            * 有效数字
                * 实部:同float
                * 虚部:同float
            * 字面值
                * 3.12455_32+654123.123J
                * -3.12455_32+654123.123j
                * 3.12455_32e2+654123.123J 
                * 3.12455_32e2+654123e2j 
                * 3.14_15_93j
                * 3.15
            * 构造复数的方法
                * 使用字面值赋值。如执行`a=5+4j`后`a`的值为`5+4j`
                * `complex([x=0j])`
                    * x:字面值或字符串表示的复数
                        * complex("-3.12455_32+654123.123j") 
                            * 注意:当从字符串转换时,字符串在 + 或 - 的周围必须不能有空格 
                        * complex(-3.12455_32 + 654123.123j) 
                        * complex(-3.12455_32+654123.123j) 
                        * complex(-3.12455_32) 
                        * complex() 
                    * 返回:复数
                * `complex(real,imag)`
                    * real:float,复数的实部
                    * imag:float,复数的虚部
                    * 返回:复数
        * bool
            * 取值范围:0,1。表示假真
            * 字面值
                * 0
                * 1
                * False,同0
                * True,同1
